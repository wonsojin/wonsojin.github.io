# 1-4. 파이썬 설치

**windows**
1. 윈도우용 파이썬 다운 (https://www.python.org/downloads/)

2. install now 선택 (설치 진행)
- add python.exe to PATH 옵션 선택 (어느곳에서든지 실행할 수 있게함)

**mac**
1. 맥용 파이썬 다운

2. 터미널에서 설치 확인
- 명령어 : pahkey@mymac ~ % python3 -V
- 결과 : python (설치 버전 확인)
